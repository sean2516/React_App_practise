# React_App_practise

***

## 功能
人员的增删查改

### 前端文件与页面
Staff_list.js - 主页面，员工信息列表.

Add_new_staff.js - 增加新的员工信息页面.

Edit_info.js - 修改当前员工的信息的页面.

Search_testing.js - 主页面的人员table的关键词搜索功能的测试原型，在完成调试后将该方法添加进Staff_list.js .
